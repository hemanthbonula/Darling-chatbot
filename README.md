# Darling Chatbot

Darling Chatbot is a simple AI-powered chatbot built using Python. It uses the ChatterBot library to learn from conversations and respond interactively with users. This project is perfect for beginners who want to explore chatbot development.

---

## 🧠 Features
- Learns from user input
- Text-based conversational interaction
- Simple GUI with Tkinter
- Easy to customize

---

## 📁 Project Structure
```bash
Darling-chatbot/
├── chatbot.py         # Main chatbot code (GUI + logic)
├── requirements.txt   # Required packages
└── README.md          # Project documentation
```

---

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/hemanthbonula/Darling-chatbot.git
cd Darling-chatbot
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Chatbot
```bash
python chatbot.py
```

---

## 🧾 Requirements
- Python 3.6 or higher
- ChatterBot
- Tkinter

You can install them using:
```bash
pip install chatterbot chatterbot_corpus
```
Tkinter usually comes pre-installed with Python. If not, install it using your system's package manager.
