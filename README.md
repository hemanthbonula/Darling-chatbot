# 💬 Darling Chatbot 💖

Darling Chatbot is a cute, emotionally supportive terminal-based chatbot designed to cheer you up, make you smile, and give you a virtual companion to talk to. Built in Python, it uses simple text-based logic to simulate a loving and caring personality.

---

## ✨ Features

- 🌸 Sweet, affectionate conversation style
- 🤗 Responds to emotions like sadness or love
- 😂 Tells you jokes to lighten the mood
- 💾 Saves your chat log after the session
- 🧠 Simple keyword-based response system
